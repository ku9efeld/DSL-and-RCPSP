Задание: тебе будет представлены пара планов проекта по бурению скважин в виде массивов, который называются следующим образом:

- Tasks activity - одномерный массив с порядком выполнения заданий. (TA).
- Resources for activity  - количество выделяемого ресурса. (RA). Последний столбец - означает id исполнителя.
- Boarder resources for contractor - максимальные границы выделяемого ресурса, где минимум -- 0. (BRC).

Сгенерируй новый план для того же самого проекта, таких, что смог минимизировать время окончания проекта. Главное ограничение - сгенерированное решение должно объединить планы из 2х предлагаемых планов.  
 
Для генерации представь матрицы  и верни сразу RA и TA в виде numpy-формата (.npz), где внутри <> начало и конец массива. Учти массивы должны быть того же размера, но с разным наполнением, что будет соответствовать новому сгенерированному плану.
